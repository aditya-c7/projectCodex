# Aditya Portfolio

This is a responsive single-page portfolio for Aditya 
## Run locally

### Option 1: Python (no dependencies)
```bash
python -m http.server 8000
```
Then open: http://localhost:8000

### Option 2: Node (npm)
```bash
npm install
npm start
```
Then open: http://localhost:8000

If you see `ERR_CONNECTION_REFUSED`, the server is not running or the port is different.
